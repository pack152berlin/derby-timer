# Sources & References

Useful references organized by topic, collected during development.

## Pinewood Derby Ranking & Scoring

- [Stan Pope - Conducting Pinewood Derbies](https://stanpope.net/pwraces.html) — Authoritative guide covering scheduling, scoring, and race management
- [GrandPrix Race Manager (GPRM) - Standings FAQ](https://grandprix-software-central.com/index.php/faqs/gprm/standings) — Tiebreaker rules and scoring options
- [GrandPrix Race Manager](https://www.grandprix-software-central.com/gprm/) — Most widely-used commercial derby software
- [DerbyNet](https://derbynet.org/) ([GitHub](https://github.com/jeffpiazza/derbynet)) — Popular open-source alternative, uses "Partial Perfect-N" scheduling
- [Maximum Velocity - Race Methods](https://www.maximum-velocity.com/pinewood-derby-race-methods/) — Elimination race method comparison
